# Operator Guide

## First Deployment
1. Register operator, stations/transmitters, broadcast-license scope,
   staff and robots.
2. Import existing program-schedule and billing history.
3. Run read-only license-scope and transmitter/studio-equipment robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run billing record and audit export.

## Minimum Production Controls
- broadcast-license-scope validation before any transmission dispatch
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (out-of-license
  transmission, public-interest-programming violations)
- evidence-backed billing records
- audit export for every dispatch, sign-off and billing record
- backup manual broadcast-operations process

## Certification
Certified operators must prove robot-safety integrity, license-scope
discipline, evidence-backed billing records and human review for
transmission-affecting actions.
